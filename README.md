# Star Alumni: Bridge Between Faculty and Students

## Project Overview

Star Alumni is a web-based platform developed to enhance communication and collaboration between students, alumni, and faculty members.

The system focuses on connecting selected top alumni with students to provide structured mentorship, conduct skill-based webinars, and offer career guidance and job referrals. It helps students gain real-world insights and supports their professional growth.

## Objectives

* To create a structured platform for alumni–student interaction
* To provide mentorship and career guidance from experienced alumni
* To organize webinars and technical sessions
* To track student progress and engagement
* To strengthen communication between faculty, students, and alumni

## Key Features

### Student Module

* View assigned alumni mentor
* Attend weekend webinars
* Get career guidance and job referrals
* Track progress after mentorship

### Alumni Module

* Conduct webinars and mentoring sessions
* Share industry experience and knowledge
* Provide job referrals
* Receive appreciation for contribution

### Faculty Module

* Monitor alumni–student interactions
* Assign alumni to student groups
* Track student performance and outcomes

## Core Functionalities

* Selection of top alumni for mentorship
* Division-wise student grouping system
* Weekend webinar scheduling (Saturday & Sunday)
* Alumni activity tracking and ranking
* Student progress monitoring system
* Alumni success stories showcase

## Installation & Setup

### 1. Clone the repository

git clone https://github.com/your-username/star-alumni.git

### 2. Navigate to project folder

cd star-alumni

### 3. Install dependencies

npm install

### 4. Start Backend Server

cd server
npm start

### 5. Start Frontend

cd client
npm start

## Environment Variables

Create a `.env` file inside the **server** folder and add:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
